# Classroom Consumption Digital Twin 

### Project Overview
This is my first digital twin project, focused on **optimising the energy consumption of a classroom**.

### Description

Thematic Area: Smart Buildings and Energy Management
Title: Development of a Digital Twin for Monitoring and Optimizing Energy Consumption in Educational Spaces.

Area/Problem: The work will focus on university classrooms. The problem is the energy waste created by the non-optimized use of lights/air conditioners within a classroom.

Added value: The proposed DT will combine both environmental and energy data to create a dynamic model of the room. It will be able to run scenarios such as: How much does consumption change if the temperature drops by 1 degree? It will be valuable for 2 reasons: 1) It does not need to be experimented on students as it will be tested first on the digital model. 2) Based on weather data, it will be able to operate predictively, finding the optimal setting before the day starts.

Tools & Methodology: The CU-BEMS (Chulalongkorn University Building Energy Management System) dataset will be used, which takes real-time time series measurements from sensors and energy meters. The methodology will include: 1) Collection and processing of time series using Python 2) Development of a Baseline Model using regression algorithms. Scenario simulations where temperature, lights and occupancy will be considered and optimal decisions will be made.

Measurable Indicators: The success of the application will be evaluated based on 1) Percentage reduction in energy consumption 2) Thermal Comfort Index (around 21-25 based on the ISO 7730 standard) 3) Accuracy of prediction of the digital model in relation to the actual data we have collected.
Future Steps: It could be interconnected with IoT sensors so that it can be controlled fully automatically. We could also use a cloud network and do it for all the rooms of a university.
